# AI News Real-Time Dashboard

A modern, high-performance web application that fetches and displays real-time AI news from ETNews.

## Features
- **Real-Time Updates**: Fetches data from ETNews AI RSS feed.
- **Modern UI**: Cyber/AI aesthetic with glassmorphism and OKLCH color spaces.
- **Web Components**: Custom `<ai-news-item>` for encapsulated styling and logic.
- **Responsive Design**: Works seamlessly on mobile and desktop.
- **Performance**: Lightweight, framework-less implementation.

## Tech Stack
- HTML5
- CSS3 (Modern features like OKLCH, Grid, Flexbox)
- Vanilla JavaScript (ES Modules, Web Components)
- [AllOrigins](https://allorigins.win/) (CORS Proxy)

## Getting Started
Simply open `index.html` in a modern web browser.
